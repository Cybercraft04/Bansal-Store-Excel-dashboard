# Bansal-Store-Excel-dashboard
Bansal Stores Sales Dashboard (Excel)
Project Overview
The Bansal Stores Sales Dashboard is an interactive Microsoft Excel dashboard designed to analyze and visualize retail sales data for the year 2022.
It transforms raw transactional data into meaningful insights using pivot tables, charts, slicers, and KPIs, enabling quick decision-making and performance tracking.

This dashboard helps stakeholders understand:
**Sales performance**
**Customer demographics**
**Product category trends**
**Sales channels**
**Geographic distribution of orders**

**Objectives**
The main objectives of this project are:

Analyze overall sales performance
Identify top-performing categories and products
Understand customer demographics
Track monthly sales trends
Compare sales channels
Visualize geographical distribution of orders

**Dataset Information**

The dataset contains order-level sales records with the following fields:

Column	Description
Order ID	Unique identifier for each order
Cust ID	Unique customer ID
Gender	Customer gender
Age	Customer age
Age Group	Age category segmentation
Date	Order date
Status	Order status (Delivered, Cancelled, etc.)
Month	Order month
Channel	Sales channel (Amazon, Flipkart, etc.)
SKU	Product identifier
Category	Product category
Size	Product size
Qty	Quantity ordered
Currency	Transaction currency
Amount	Order value
Ship City	Shipping city
Ship State	Shipping state
Ship Postal Code	Postal code
Ship Country	Shipping country
B2B	Indicates if the order is B2

▶️ How to Use

Open the Excel file.

Navigate to the Store Dashboard sheet.

Use the slicers to filter data by category, month, or channel.

Charts and KPIs will update automatically based on the selected filters.
